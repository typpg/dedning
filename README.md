# ミライース・デッドニング施工マニュアル

## 1. 資材・道具リスト

### 【購入資材】
* レアルシルト ハーフ (4枚入) ×1
* オーディオテクニカ AT-AQ470 (吸音材) ×1
* オーディオテクニカ AT-AQ450 (制振材/穴塞ぎ) ×2
* 東レ ペフシート (1m×1m / 5mm厚) ×1
* パーツクリーナー (840ml 大容量タイプ) ×2本

### 【持参ツール】
* アルミテープ、マスキングテープ
* 圧着ツール、ヘラ、内張り剥がし
* シリコンオフ
* ニトリル手袋、ゴミ袋
* ハさみ、カッター
* ウエス (Tシャツ4枚程度 / キッチンペーパー併用推奨)

## 2. 資材の用途とはる位置

| 資材名 | 施工場所 | 主な役割 |
| :--- | :--- | :--- |
| **レアルシルト** | アウターパネル（ドア奥） | 鉄板の振動を抑え、重厚な音にする |
| **AT-AQ470** | アウターパネル（スピーカー裏） | スピーカー背面の音を吸収し濁りを取る |
| **AT-AQ450** | インナーパネル（サービスホール） | 大きな穴を塞ぎ、ドアを密閉箱にする |
| **ペフシート** | インナーパネル（全面） | ロードノイズを遮断し、断熱効果を高める |
| **アルミテープ** | 各所の端・隙間 | 制振材の剥がれ防止とワイヤー部の密閉 |

## 3. 施工工程表（片側目安：約2.5時間）

### ① 解体・清掃（約60分）
1. ドアポケットとハンドルのネジ（計2本）を外す。
2. 内張り剥がしでパネルを外し、カプラーを抜く。
3. 純正ビニールを剥がし、ブチルゴムをヘラで削ぎ落とす。
4. **【重要】**パーツクリーナーで鉄板を脱脂（キュッキュッとなるまで）。

### ② アウターパネル施工（約30分）
1. 10cm角に切ったレアルシルトを、叩いて響く場所に分散して貼る。
2. スピーカーの真裏にAT-AQ470（吸音材）を貼る。
3. **【必須】**窓を全開まで下げ、ガラスが資材に当たらないか確認。

### ③ インナーパネル施工（約45分）
1. 大きな穴（サービスホール）をAT-AQ450で塞ぎ、ヘラで圧着する。
2. ドアノブワイヤーが出る部分はスリットを入れ、アルミテープで隙間を埋める。
3. 端が浮きやすい場所をアルミテープで補強する。

### ④ ペフシート仕上げ（約30分）
1. 50cm×100cmに切ったペフシートを上から順に空気を抜いて貼る。
2. **【重要】**クリップ穴、ネジ穴、ワイヤー出口をカッターでくり抜く。

### ⑤ 復元（約15分）
1. カプラーを繋ぎ、スピーカーの音出しとパワー窓の動作を確認。
2. 内張りを「パチン」と音がするまで叩いてハメ込み、ネジを締める。

---

## 4. 失敗しないための3大鉄則

1. **脱脂は執拗に：** ブチルが残っていると、夏場にすべて剥がれ落ちます。
2. **ワイヤーの自由度：** ドアロックワイヤーが制振材に挟まって動かなくならないよう注意。
3. **排水口を塞がない：** ドア最下部にある小さな「水抜き穴」は絶対に塞がないこと。
